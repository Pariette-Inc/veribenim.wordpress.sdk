# veribenim/wordpress

> Veribenim KVKK & GDPR çerez onayı — WordPress Plugin

[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](https://opensource.org/licenses/MIT)

## Kurulum

**WordPress Eklenti Dizininden:**
1. WordPress yönetici paneli → Eklentiler → Yeni Ekle
2. "Veribenim" arayın ve yükleyin
3. Etkinleştirin

**Composer ile:**
```bash
composer require veribenim/wordpress
```

## Kurulum Sonrası

1. **Ayarlar → Veribenim** sayfasına gidin
2. [Veribenim Paneli](https://app.veribenim.com)'nden aldığınız **Site Token**'ı girin (Siteniz → Entegrasyon)
3. Dil tercihini seçin
4. Kaydedin — banner scripti tüm sayfalara otomatik eklenir

## Özellikler

- Otomatik script injection — `<head>` içine eklenir
- Admin panel entegrasyonu — token ve dil tek sayfadan yönetilir
- Tema bağımsız, WooCommerce uyumlu
- Multisite desteği

## Gereksinimler

- PHP 8.1+
- WordPress 6.0+
